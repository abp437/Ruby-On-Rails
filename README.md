# Dependencies:

## Ruby - 2.4.4
## Rails - 5.2.1
## Node - 8.11.3(LTS)
## React - 16.5.1
## Postgres - 10.4

# Server Setup:

* cd server
* bundle
* rails db:create
* rails db:migrate

# Client Setup:

* cd client
* npm install

# Running:

### Client:

* cd client
* npm start

### Server:

* cd server
* rails s
